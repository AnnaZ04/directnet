DirectNet Website

-- A modern responsive website built with React and Tailwind CSS for a cleaning services company.

-- https://directnet-4ow9.vercel.app

-- About the project:

This project is a landing page for a cleaning services company called DirectNet.  
The goal was to create a clean, modern and responsive website that works well on all devices and provides clear information about services, advantages and contact details.

The design focuses on simplicity, readability and user experience.


-- Tech Stack:

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Hook Form
- Vitest (for testing)

-- Features:

- Fully responsive design (mobile, tablet, desktop)
- Smooth animations using Framer Motion
- Contact form 
- Interactive UI components
- Modular and reusable components
- Basic unit tests using Vitest

--Sections:

- Navbar with smooth navigation
- Hero section
- About section with statistics
- Services section
- Company methodology section
- Contact form with validation
- Footer

-- Testing:

Basic tests were implemented using Vitest and React Testing Library to ensure form functionality and UI rendering.

-- During this project I improved my skills in:

- Building responsive layouts with Tailwind CSS
- Managing component structure in React
- Handling forms with React Hook Form
- Adding animations with Framer Motion
- Writing basic frontend tests
- Working with Git and GitHub workflow
- Deploying projects on Vercel



-- Deployment:

The project is deployed using Vercel and automatically updates after each push to GitHub.
